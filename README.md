# 👟 BLYND — Advanced Shoe Brand Website (Nike-Inspired)

A sleek, responsive website for a futuristic footwear brand — inspired by Nike’s premium design style.  
Built with pure HTML, CSS, and JavaScript for smooth animations, bold visuals, and lightning-fast performance.

---

## 🚀 Features

- ⚡ **Modern UI/UX:** Nike-style clean design with bold typography and gradients  
- 📱 **Fully Responsive:** Works perfectly across all devices  
- 🛒 **Product Cards:** Hover effects, quick-view modal & add-to-cart mockup  
- 🎨 **Dark Futuristic Theme:** Glassmorphism & neon accents  
- 🧠 **Lightweight & Fast:** Pure HTML/CSS/JS — no frameworks  
- 💬 **Waitlist Form:** Users can join your brand drop list  

---

## 🛠️ Tech Stack

| Tech | Purpose |
|------|----------|
| **HTML5** | Structure & layout |
| **CSS3** | Styling & responsive design |
| **JavaScript (ES6)** | Interactivity & animations |
| *(Optional)* Tailwind / GSAP | For extended design & animation |
